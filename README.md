基于Node.js的多人博客

## Installation

### Prerequire

* Node.js >= v10.13.0
* mongodb v3
* redis v2.6
* Git >= 2.13.0 (for git precommit hook)

### Run

* Install packages `npm install`
* Build `npm run build`
* Development start `npm run dev`
* Start `npm start`
* Build and restart `npm restart`
* Restart `make reboot`
* Open `http://localhost:3001` in browser

## test

* `npm test`

## coverage

* `npm run test-cov`
